Common questions

Here are some of the most common questions that we get.
What is the purpose of the NASA Hunch Network Security Design? The purpose of the NASA Hunch Network Security Design is to securely share files and data within the NASA network without data loss due to environmental factors in Space.
Why use the NASA Hunch Network Security Design? The current issues with data corruption through cosmic/gamma rays: The first step to data integrity is being aware of environmental issues BEFORE transferring files. In outer space, the transfer of data can become corrupted by astronomical objects that have a changing magnetic field, such as the planets and our Sun.
How secure is the NASA Hunch Network Security Design As an extra layer of security, the different devices that carry out the Network Security Design each have different levels of clearance and permissions ranging from a Level 1-4. Thus, lower chances of a hack.
How fast can the NASA Hunch Network Security Design share data? The NASA Hunch Network Security Design prototype can currently send about 500 Megabytes per second (will be even faster).
